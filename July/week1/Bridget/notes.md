# JavaScript Simple Calculator

## Objective
Build a simple calculator that performs basic arithmetic operations using JavaScript.

## Concepts Used
- Variables (`let`, `const`)
- Data Types (Number, String)
- Operators (`+`, `-`, `*`, `/`)
- `console.log()`
- Basic Input/Output (`prompt()`)

## Steps
1. Get the first number from the user.
2. Get the second number from the user.
3. Ask the user to choose an operator (`+`, `-`, `*`, `/`).
4. Perform the selected calculation.
5. Display the result in the console.

## Example
Input:
- First Number: 10
- Second Number: 5
- Operator: +

Output:
```
10 + 5 = 15
```
